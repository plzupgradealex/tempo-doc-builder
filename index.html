Getting Started

Tempo helps you build professional agendas for client visits.

  • Fill in the project header with client info
  • Answer pre-work questions to generate requirements
  • Add travel details for arrival and departure
  • Add topics to each day from the knowledge domain picker
  • Drag & reorder events within each day
  • Preview & export as PDF for emailing or printing

Keyboard Shortcuts

  • Ctrl+S — Save agenda
  • Ctrl+P — Preview PDF
  • Ctrl+E — Export PDF

Star Trek Mode

Toggle the MODE button in the top bar to switch between Standard and Star Trek mode. Trek mode changes UI labels but never affects your PDF exports.
